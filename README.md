# china-unicom-internet-access-analyze
 联通上网记录详单分析程序

## 用法

 php analytic.php filename.csv

 php analytic.php filename.csv 流量偏移量 IP偏移量 访问网址偏移量

## 依赖

 php5以上

## 特别说明
